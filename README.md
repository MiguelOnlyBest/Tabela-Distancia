### Descrição

O Projeto consite na composição de uma tabela que conta com capitais brasileiras e distância entre essas capitais. As capitais constantes nesse projetos são:

* Manaus
* Natal
* Palmas
* Porto Alegre
* Porto Velho
* Recife 
* Rio Branco
* Rio de Janeiro
